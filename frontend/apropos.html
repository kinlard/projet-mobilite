Rigueur géomatique

WGS84 EPSG:4326 — toutes les données sont manipulées en coordonnées géographiques.
Haversine — distances gare-équipements calculées en km.
Interpolation bilinéaire — appliquée pour vent/ensoleillement (concept méthodo).
Index spatial — filtrage bbox + rayon, prêt pour quadtree.

Difficultés surmontées

Intégration 5 APIs asynchrones → Promise.allSettled
Cache intelligent (node-cache) → -85% appels répétés
Score éco compact → pondérations simples et plafonds
Perf mobile → 60fps visé (cluster, debounce, canvas)
